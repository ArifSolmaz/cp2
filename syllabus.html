üìñ Course Description

This course builds upon the fundamentals of programming learned in Computer Programming I. Students will develop intermediate to advanced Python programming skills, focusing on object-oriented programming, file handling, error management, and working with external data sources. The course emphasizes practical applications relevant to mechatronics engineering, including data processing, automation scripts, and system integration.

Through hands-on exercises and a comprehensive final project, students will gain the skills necessary to develop robust, maintainable Python applications for engineering contexts.

üéØ Learning Objectives

Upon successful completion of this course, students will be able to:

  1. Write well-structured Python programs using functions, classes, and modules
  2. Apply object-oriented programming principles including inheritance and polymorphism
  3. Implement proper error handling and defensive programming techniques
  4. Read from and write to various file formats (text, CSV, JSON)
  5. Work with external libraries and APIs for data acquisition
  6. Design and implement a complete software project from requirements to deployment
  7. Debug and test Python code effectively
  8. Apply Python programming to solve mechatronics engineering problems

üìú Course Policies

Attendance

Attendance is mandatory. Students must attend at least 70% of classes to be eligible for the final exam. Weekly exercise submissions are tracked as part of attendance.

Late Submissions

Weekly exercises must be submitted during the designated submission window. Late submissions are not accepted unless prior arrangements are made with the instructor.

Academic Integrity

All work submitted must be your own. Collaboration is encouraged for learning, but submitted code must be individually written. Plagiarism detection tools are used to check submissions. Violations will result in a zero for the assignment and potential disciplinary action.

Communication

Use your @istun.edu.tr email for all course-related communication. Questions about course material can also be asked during office hours or through the course platform.
