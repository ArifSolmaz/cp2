‚å®Ô∏è Part 10: User Input
üí° Note: The input() function doesn't work in this browser environment. The examples below show the syntax - practice these in Google Colab or a local Python environment! 
# Basic input - always returns a STRING
name = input("Enter your name: ")
print(f"Hello, {name}!")

# Converting input to numbers
age = int(input("Enter your age: "))
print(f"Next year you'll be {age + 1}")

height = float(input("Enter your height (m): "))
print(f"Height in cm: {height * 100}")

# One-liner with conversion
weight = float(input("Enter weight (kg): "))
bmi = weight / (height ** 2)
print(f"Your BMI: {bmi:.2f}")
