Use your phone stopwatch + Python to estimate vertical climbing power.

Week 01 — Instructions

  1. Identity: Enter Student ID (6–12 digits), Name Surname, and @istun.edu.tr email.
  2. Measure: Choose stairs. Count steps (N_steps) and estimate step height (step_height_m). Measure time 5 trials using phone stopwatch.
  3. Enter Data: Fill mass_kg, N_steps, step_height_m, and t=[...].
  4. Run Python: Check output (HEIGHT, AVG_TIME, AVG_POWER).
  5. Submit: One submission only. A new tab opens for “Accepted / Rejected / Duplicate”.
