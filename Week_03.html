Introduction to Loops

Loops allow you to execute a block of code multiple times. They are fundamental to programming and essential for:

  • üîÅ Automating repetitive tasks - Perform the same operation many times
  • üìä Processing collections of data - Work with lists, strings, and other sequences
  • üî¢ Performing calculations iteratively - Sum, average, factorial, etc.
  • üé® Creating patterns and sequences - Generate structured outputs
üí° Two Types of Loops

for loop: Iterate over a sequence (known number of iterations)
while loop: Repeat while a condition is true (unknown number of iterations)

Loop Type Use When Example
for Number of iterations is known Processing all items in a list
while Number of iterations is unknown Waiting for user input

The for Loop

The for loop iterates over items in a sequence (list, string, range, etc.).

Basic for Loop Syntax Python 
for item in sequence:
    # Code to execute for each item
    print(item)

The range() Function

The range() function generates a sequence of numbers, commonly used with for loops.

Syntax Description Example Output
range(stop) 0 to stop-1 range(5) 0, 1, 2, 3, 4
range(start, stop) start to stop-1 range(2, 6) 2, 3, 4, 5
range(start, stop, step) start to stop-1, with step range(0, 10, 2) 0, 2, 4, 6, 8
‚ö†Ô∏è Important

The stop value is exclusive - it's NOT included in the sequence!

range_examples.py 
# range(stop) - 0 to stop-1
print("range(5):")
for i in range(5):
    print(i, end=" ")

print("\n\nrange(1, 6):")
for i in range(1, 6):
    print(i, end=" ")

print("\n\nrange(0, 11, 2) - even numbers:")
for i in range(0, 11, 2):
    print(i, end=" ")
Output

Using enumerate()

The enumerate() function returns both the index and value when iterating.

enumerate_example.py 
# Using enumerate to get index and value
fruits = ["apple", "banana", "cherry", "date"]

print("Default (start=0):")
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

print("\nWith start=1:")
for num, fruit in enumerate(fruits, start=1):
    print(f"{num}. {fruit}")
Output
üí° When to Use enumerate()

Use enumerate() when you need both the index and the value. It's cleaner than manually tracking an index variable!

The while Loop

The while loop continues executing while a condition is True.

while Loop Syntax Python 
while condition:
    # Code to execute while condition is True
    # Don't forget to update the condition!
‚ö†Ô∏è Infinite Loop Warning!

Make sure the condition eventually becomes False, or you'll have an infinite loop that never stops!

while_basic.py 
# Basic while loop
count = 1

while count <= 5:
    print(f"Count: {count}")
    count += 1  # Don't forget this!

print("Loop finished!")
Output

Loop Control: break and continue

Statement Effect Use Case
break Exits the loop entirely When target is found
continue Skips to next iteration Skip unwanted items

The else Clause with Loops

The else block executes when the loop completes normally (without break).

üí° Key Concept

The else block runs when the loop finishes all iterations. If break is executed, the else block is skipped!

loop_else.py 
# Search with else clause
print("Searching for 4 in [1,3,5,7,9]:")
numbers = [1, 3, 5, 7, 9]
target = 4

for num in numbers:
    if num == target:
        print(f"Found {target}!")
        break
else:
    print(f"{target} not found in list.")

print("\nSearching for 5 in [1,3,5,7,9]:")
target = 5
for num in numbers:
    if num == target:
        print(f"Found {target}!")
        break
else:
    print(f"{target} not found in list.")
Output

    print()  # New line after inner loop

Sensor Data Processing

sensor_average.py 
# Calculate average voltage from sensor readings
# Skip invalid readings (-999)
readings = [11.8, 12.1, -999, 12.3, 12.0, -999, 12.2]

valid_sum = 0
valid_count = 0

for reading in readings:
    if reading == -999:
        continue  # Skip invalid reading

    valid_count += 1

average = valid_sum / valid_count
print(f"Valid readings: {valid_count}")

# Newton-Raphson method to find square root
number = 25
guess = number / 2
tolerance = 0.0001
iteration = 0

while abs(guess * guess - number) > tolerance:
    guess = (guess + number / guess) / 2
    iteration += 1
    print(f"Iteration {iteration}: {guess:.6f}")

print(f"\n‚àö{number} ‚âà {guess:.6f}")

Find Prime Numbers

find_primes.py 
# Find all prime numbers up to n
n = 30
print(f"Prime numbers up to {n}:")

for num in range(2, n + 1):
    is_prime = True
    for i in range(2, int(num ** 0.5) + 1):

Summary

Concept Syntax Use Case
for loop for item in sequence: Known iterations, sequences
range() range(start, stop, step) Generate number sequences
enumerate() for i, v in enumerate(seq): Get index and value
while loop while condition: Unknown iterations, conditions
break break Exit loop early
continue continue Skip to next iteration
Loop else else: after loop Runs if no break
Nested loops Loop inside loop 2D data, patterns
üéØ Key Takeaways
  • Use for when you know how many times to iterate
  • Use while when iteration depends on a condition
  • Always ensure while loops can terminate
  • Use break to exit early, continue to skip
  • Master common patterns: accumulator, counter, filter, transform
