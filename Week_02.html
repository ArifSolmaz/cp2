The if-elif-else Chain

When you have multiple conditions to check, use elif (else if). This creates a chain where only one block executes.

Syntax

if condition1:
    # Code if condition1 is True
elif condition2:
    # Code if condition2 is True
elif condition3:
    # Code if condition3 is True
else:
    # Code if all conditions are False

Key Point: Python checks conditions top to bottom and stops at the first True condition. Only one block executes.

Common Mistakes to Avoid

These are the most frequent errors students make with control flow. Study them carefully ‚Äî recognizing these patterns will save you hours of debugging!

Using = instead of ==
if x = 5: ‚Üí SyntaxError! Use if x == 5:
Remember: = assigns a value, == compares two values.

Forgetting the colon :
if x > 10 ‚Üí SyntaxError! Every if, elif, and else line must end with a colon :

Incorrect indentation
All code inside an if-block must be indented by the same amount (4 spaces). Mixing tabs and spaces causes IndentationError. Configure your editor to use spaces.

Using elif after else
else must always be LAST in the chain. You cannot write elif after else ‚Äî Python will raise a SyntaxError.

Comparing different types without conversion
if input("Age: ") > 18: ‚Üí This compares a string to an int, which always gives unexpected results!
Fix: if int(input("Age: ")) > 18:
