üîÄ Part 4: The if-elif-else Statement

üéØ Learning Objectives

  • Use elif for multiple conditions
  • Create multi-way decision structures
  • Build grade calculators and category systems

Use elif (short for "else if") when you have more than two possible outcomes.

üìå Syntax: 
if condition1:
    # Runs if condition1 is True
elif condition2:
    # Runs if condition1 is False AND condition2 is True
elif condition3:
    # Runs if condition1 and condition2 are False AND condition3 is True
else:
    # Runs if ALL conditions are False
